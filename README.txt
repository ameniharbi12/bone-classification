# Bone Classification

A deep learning project for bone image classification.

## 📌 Overview
This project trains a CNN model (ResNet50) to classify bone X-ray images into categories: fractured vs not-fractured.

## 📂 Project Structure
- `bone_classification.ipynb` → main Colab notebook
- `requirements.txt` → dependencies
- `src/` → helper functions (optional)
- `data/` → sample dataset or links to dataset (if large)

## ⚙️ Installation
```bash
pip install -r requirements.txt
